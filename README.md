# MyVMK Monthly Prize Tracker

A simple browser-based tracker for MyVMK Chest Keys and SITS prizes.

**Live Site:** https://bsims-codes.github.io/myvmk-monthly-prize-tracker/

## Features

- Track Chest Key prizes (Bronze, Silver, Gold)
- Track SITS prizes (Common, Rare, Ultra), Credits, and Ash
- Monthly prize configurations updated by bsims
- Separate summaries for Keys and SITS with detailed breakdowns
- Win rate and ash rate statistics per key color
- "All Months" view for aggregate stats across all time
- Dark/Light mode toggle
- Export/Import data as JSON

## Usage

1. Go to https://bsims-codes.github.io/myvmk-monthly-prize-tracker/
2. Select the month and system (Chest Keys or SITS)
3. Click prize buttons to log wins, or add Credits/Ash
4. View your statistics in the summary sections

## Data Storage

All data is stored locally in your browser's localStorage - there are no accounts or servers. **Export regularly** to back up your data, as clearing browser data will erase your history.

## Monthly Prize Updates

Prize configurations are updated each month by bsims. If prizes aren't showing for the current month, check back soon or reach out to bsims.
